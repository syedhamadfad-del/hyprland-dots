# Hyprland Dots

This repository contains configuration files for Hyprland, including setups for Waybar, Rofi, and the Hyprland window manager itself.

## Installation Instructions

### Prerequisites
- Ensure you have Hyprland installed on your system. You can follow the installation guide from the [official Hyprland website](https://hyprland.org/).

### Clone the Repository
To get the configuration files, clone the repository:
```bash
git clone https://github.com/syedhamadfad-del/hyprland-dots.git
cd hyprland-dots
```

### Waybar Configuration
1. Place the Waybar configuration files in the appropriate directory, typically `~/.config/waybar/`.
2. Example setup:
   - Copy `waybar/config` to `~/.config/waybar/config`
   - Copy `waybar/style.css` to `~/.config/waybar/style.css`

### Rofi Configuration
1. Place the Rofi configuration files in `~/.config/rofi/`.
2. Example setup:
   - Copy `rofi/config.rasi` to `~/.config/rofi/config.rasi`

### Hyprland Configuration
1. Place the Hyprland configuration files in the `~/.config/hypr/` directory.
2. Example setup:
   - Copy `hypr/hyprland.conf` to `~/.config/hypr/hyprland.conf`

## Usage

After placing all configuration files in their respective directories, start or restart Hyprland to apply the new settings.

## Customizations

Feel free to customize the configuration files according to your preferences. Each config file is well-commented to guide you through possible changes.
